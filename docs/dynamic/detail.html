# 获取动态详情

https://api.bilibili.com/x/polymer/web-dynamic/v1/detail

请求方式：GET

是否需要登录：否

# URL参数

参数名类型必填内容备注
timezone_offsetnum-480
idnum√动态id

# Json回复

# 根对象

字段名类型内容备注
codenum响应码0：成功
4101139：4101139
messagestr
ttlnum1
dataobj信息本体

# data对象

字段名类型内容备注
itemobj

# data对象 -> item对象

参照 获取动态列表

# 请求示例

curl -L -X GET 'https://api.bilibili.com/x/polymer/web-dynamic/v1/detail?id=724328028268658744'
